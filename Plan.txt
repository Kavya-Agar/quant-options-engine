OPTIONS PRICING ENGINE + GREEKS DASHBOARD — PROJECT PLAN

STACK
- Backend: Python, FastAPI
- Math: NumPy, SciPy
- Data: yfinance, FRED API
- Frontend: React, Recharts or Plotly.js
- Extras: Docker (optional, looks good on resume)

ARCHITECTURE
Data Layer:  yfinance (options chain + spot price)
             FRED API (risk-free rate)
             Historical volatility (rolling std of log returns)
Engine:      Black-Scholes closed-form pricer
             Monte Carlo GBM path simulator
             IV solver (Brent's method)
             Greeks calculator (Delta, Gamma, Vega, Theta, Rho)
Dashboard:   FastAPI REST endpoints
             React frontend (ticker input, strike/expiry selector)
             Greeks heatmap, P&L chart, IV surface

---

PHASE 1 — Core Math (Days 1-3)
- Implement Black-Scholes from scratch for calls and puts
- Use scipy.stats.norm for the CDF (N(d1), N(d2))
- Derive each Greek analytically:
    Delta = N(d1) for calls
    Gamma = derivative of delta
    Vega, Theta, Rho from BS formula derivatives
- Test against known reference values before moving on
- No UI yet — pure Python scripts only

PHASE 2 — Monte Carlo Pricer (Days 4-5)
- Simulate geometric Brownian motion paths (10,000-50,000 simulations)
- Price option as discounted average payoff across paths
- Compare output to Black-Scholes — they should converge
- Add antithetic variates for variance reduction
- Document the convergence comparison (good talking point in interviews)

PHASE 3 — Market Data + IV Solver (Days 6-8)
- Use yfinance to pull a real options chain (SPY or AAPL recommended)
- Pull 3-month T-bill rate from FRED API as risk-free rate (free, no key needed)
- Build implied volatility solver:
    Given a market price, find sigma such that BS(sigma) = market price
    Use scipy.optimize.brentq (Brent's method — more robust than bisection)
- Compute IV for every contract in the options chain

PHASE 4 — FastAPI Backend (Days 9-10)
- Endpoint 1: returns theoretical price + all Greeks given (ticker, strike, expiry)
- Endpoint 2: returns full options chain with IV computed per contract
- Add basic input validation and error handling

PHASE 5 — React Dashboard (Days 11-14)
- Greeks panel: sliders for spot price, time to expiry, volatility
  → updates Delta/Gamma/Vega/Theta in real time
- Heatmap: strike on x-axis, expiry on y-axis, colored by IV or Delta
- Mispricing chart: scatter plot of theoretical vs. market price per contract
